# Product Overview

This is a personal developer portfolio website for Nicholas McGuffin, a software engineer. The site showcases professional skills, work experience, and projects in a clean, modern interface.

## Key Features
- Single-page application with smooth scroll navigation
- Sticky sidebar with navigation links and social media contacts
- Three main content sections: About, Experience, and Projects
- Responsive, mobile-friendly design
- SEO-optimized with proper metadata

## Target Audience
Recruiters, potential employers, clients, and professional contacts looking to learn about Nicholas's technical capabilities and work history.

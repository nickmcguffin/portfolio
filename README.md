# Portfolio

This is my personal developer portfolio, built to showcase my professional skills, experience, and projects. It highlights the work I’ve done as a software engineer and provides an overview of my technical capabilities.

## Tech Stack
- Next.js
- React
- TypeScript
- TailwindCSS

## Features
- Clean, responsive layout
- Sticky sidebar navigation (About, Experience, Projects, Contact)
- Fast, SEO-friendly pages using Next.js App Router
- Components built with modern React patterns
- Fully mobile-friendly design

## Getting Started

Install dependencies:

```bash
npm install
```

Start the dev server:
```bash
npm run dev
```
The site will be available at:
```bash
http://localhost:3000
```

## Deployment
This project can be deployed to any platform that supports Next.js, such as:
- Vercel
- Netlify
- Github Pages (with static export)
- Your own server
